Getting MAS by HTTP

This directory contains MAS, the Modula-2 Algebra System, Version 1.0
  • The announcement of version 1.00.
  • README for version 1.00.
  • MAS source code (browse source).
  • MAS binary for HPPA running HPUX.
  • MAS binary for intel PC running OS/2, MS-DOS or MS-Windows. For DOS or Windows you will also need EMX or RSX.
  • MAS binary for intel PC running LINUX.
  • MAS multi-architecture binary for Nextstep.
  • MAS binary for RS/6000 running AIX 3.
  • MAS binary for Sun Sparc running SunOS.
  • MAS - Interactive Usage (PDF, PostScript, LaTeX source).
  • MAS - Specifications, Definition Modules, Indexes (PDF, PostScript, LaTeX source).
  • Compiling, porting and modifying MAS (PDF, PostScript, Texinfo source).
  • Send remarks, bug-reports, questions about MAS to kredel@rz.uni-mannheim.de
Patches for programs neccessary to compile MAS:
  • Patches for the Makemake program.
  • Patches for the Modula-2 to C translator MTC.
  • Patches for the Reuse library.
